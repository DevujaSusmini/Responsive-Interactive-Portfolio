Projects

Project 1

This webpage is an educational site about animals, featuring informative sections on birds, fish, and insects. Each section provides a detailed description of the animal group, accompanied by relevant images—such as an owl for birds and an underwater scene for fish. The page also summarizes the six main types of animals in the Animalia kingdom: mammals, reptiles, amphibians, birds, fish, and insects. At the end, there is a highlight on the Sinharaja Forest, including an embedded Google map and a YouTube video, showcasing it as a rich habitat for animal diversity
Features Developed:
- HTML for structuring the content with headings, paragraphs, lists, and images.
- CSS for styling the page, including setting the background color and arranging images and text.
- Image Embedding using the tag to display animal photos.
- Hyperlinks with the tag for "click for further details" links.
- Google Maps Embed to show the location of the Sinharaja Forest.
- YouTube Embed for including a video about the forest.


Project 2

This project was developed to raise awareness about Sustainable Development Goal 6 (Clean Water and Sanitation) using a fully responsive, accessible, and user-friendly website. As the editor and developer, I implemented a multi-page site that includes educational content, volunteer engagement, and real-world data representation.
Features Developed:
- Splash Screen: Designed a visually engaging splash screen with responsive layout and semantic HTML5 structure to introduce users to the project.
- Volunteer Page: Built an interactive volunteer section with card-based layout, filtering options, and a registration form for community participation.
- Water Access Content Page: Showcased global water scarcity issues using structured text, images, and accessibility-focused layout.
- Navigation & Routing: Implemented consistent navigation across all pages with Font Awesome icons and a mobile-friendly hamburger menu.
- Editor Page: Added an “Editor” profile page displaying project contribution details with validation results and compliance badges.
